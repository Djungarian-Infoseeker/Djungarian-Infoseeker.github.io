# 学校介绍
东京工业大学（日语：東京工業大学；英语：Tokyo Institute of Technology），位于东京都目黑区与横浜市绿区。创设于1881年，是一所拥有140多年历史的、专攻工程技术与自然科学的日本高等理工类研究型国立大学。是日本国大学实力第一梯队“东京一工”（东京大学、京都大学、一桥大学、东京工业大学）其中一员。由于工科可以说是日本天花板存在，就业质量甚至可以超过东京大学位列第一。硬伤是在大众不太出名（企业和学界会很懂），日本和国内都是。

其前身是1881年以培养工学方面专业人才为目的而建立的“东京职工学校”。1929年，其被提升为国立大学。2023年QS世界大学排名中东京工业大学排行全球第55位。东京工业大学在2024年10月1日和东京医科齿科大学合并为东京科学大学，实现强强联合（拿的是华中理工大学和同济医科大学的剧本），

# 护照和签证
各大省市的护照申请可以在各个省市的政务服务小程序上找相应的办事流程。在上海地区办护照会很快，我当时只用了一周就收到了新护照。拿到了护照和内推资格之后，东京工业大学会给你发相应的邮件引导你去填写信息。这里我就不再赘述。我想多说一点官方文件很少的。
