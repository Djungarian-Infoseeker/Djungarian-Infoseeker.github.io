# 推荐免试的准备

## 最重要！！！确定研究方向
在推免过程中，最重要的第一步是明确自己的未来发展方向。你需要对自己感兴趣的大致领域有一个较为清晰的认知，这些大方向的思考最好在大二时就开始。到了大三，尤其是上学期，进一步缩小范围，明确自己想要深入研究的具体领域。

我个人的建议是，在大三上学期结束前完成对自己未来研究方向的确认，并且对可能的目标院校、导师及相关的推免流程进行详细的调查和整理。这一步可以通过汇总信息并制成一个表格来实现，这样能够更加系统化地管理推免的准备工作，确保没有遗漏。

与此同时，建议大家在了解各位导师的研究方向时，保持开放的心态，或许会发现一些自己此前未曾接触过、但却非常契合个人兴趣的领域。在这个过程中，可以通过反复思考、内化，并通过交互式的分析与研判，进一步厘清自己的研究方向。总结出适合自己的领域后，可以对每位导师的研究方向逐一进行深入思考。特别是对于那些你兴趣最为浓厚的导师，建议花更多时间研究他们的学术风格、发表的论文以及他们的招生偏好。这不仅有助于更好地理解他们的研究理念，还能帮助你判断自己是否与其研究方向和风格契合，从而更有针对性地准备推荐免试的相关材料。

## 联系导师
除了平时老掉牙说的邮件套磁导师之外，我的建议是可以在本科阶段多参加学术活动，听取老师们的报告进行了解。尤其是可以提前一两年在线下与老师多交流沟通。这样经过多了解、混脸熟之后，选择的导师应该更适合自己，被心仪导师接受的概率也更高。我认为这一步很重要，不能把高中那种做题家光做题乱卷就为个高分，然后考了高分要干嘛一问三不知的毛病带到大学。

## 材料准备、应试准备
在大三寒假和大三下学期开学时，是准备推荐免试研究生的关键阶段。此时，可以开始系统地整理和准备各类申请材料。首先是文书材料的准备。你需要提前整理好所有的获奖材料，包括原件和复印件。一般来说夏令营都需要提交两份副教授以上署名的推荐信，建议准备两封不同侧重点的推荐信。

另外一般需要提交自我陈述，建议准备两份不同长度的版本。一份1500字的详尽自述，用来深入展示你的学术背景、研究经历和未来的研究规划；另一份800字的简短自述，用于简要概述你的主要成就与研究兴趣。

在科研与专业知识准备方面，针对你所申请的导师和研究方向。结合自己的研究兴趣，提前构思一些学术问题，展示你对该领域的理解。同时，也要准备常见的面试问题，例如“为什么选择该导师或领域？”、“你的研究优势是什么？”等。

最后，面试时所需的资料也要提前准备好。通常，学校或导师会要求你在面试时展示科研成果或研究计划，因此建议准备一个简洁明了的PPT，突出你的研究背景、方法、成果和未来规划。此外，提前准备好一份流畅的自我介绍稿，分别用中文和英文各准备一份。自我介绍时，尽量简洁明了，尽可能结构性的、有说服力的突出你的学术兴趣、研究经历以及与你所申请的方向的契合度。

### 面试指南：以考代练
在大学生活中，许多学生很少有机会接触正式的面试场景，这使得在面对推免或夏令营等重要面试时，可能会感到紧张或缺乏经验。为了克服这种情况并提升相关能力，我的建议是通过实际参与的方式逐步积累面试经验，做到“以考代练”。以下是一些具体建议：

#### 1. 主动参与冬令营或其他相关活动

即使某些冬令营并非你的目标院校或方向，也可以积极报名参加，将其视为锻炼面试能力的机会。通过多轮的模拟和正式面试，你可以逐渐熟悉面试流程，了解评委的关注点，从最初的紧张不安到逐步从容应对，最终甚至可以成为应对面试的“高手”。这种积累不仅能提高你的表达能力和逻辑思维，还能帮助你总结适合自己的面试策略。

> 插一句嘴，这让我想起刚上大学时遇到的 presentation 课程。当时特别害怕上台展示，经常负责做 PPT，而 PPT 质量也一般，大多是照着稿子念。这样一来，不仅自己紧张，老师也很难产生兴趣。后来随着练习次数的增加，逐渐培养了胆量和经验，发现其实 presentation 没那么可怕，就像和老师聊天一样自然。随便拉拉扯扯聊些内容，反而更容易让观众听懂，也更受欢迎。这种体验也让我意识到，熟悉与自然的表达方式比“完美”更重要。

#### 2. 提前提升英语听说能力

考虑到大多数院校在夏令营或推免环节中会重点考察英语听说能力，我建议在大三下学期开学时便着手准备语言考试。例如，参加雅思考试并尽可能争取一个高分。这不仅能够作为一份有力的英语资历材料提交，还能显著提升你的听说能力，帮助你更自信地应对面试中可能出现的英语交流或自我介绍环节。相比四六级考试，雅思更能反映实际的语言运用能力，尤其是在听力和口语方面，这对于夏令营或面试中表现出良好的英语沟通能力尤为关键。

#### 3. 综合训练应对心理

多次参与面试或类似情境后，你会逐步适应并消除对面试的恐惧心理。语言能力的提升与对面试流程的熟悉相结合，不仅可以让你更加从容地面对各种问题，还能通过自信的表现赢得面试官的认可。

通过不断实践与总结，面试能力和英语水平的提高将为你的研究生推免或夏令营申请提供有力支持，同时也为未来的职业发展奠定扎实的基础。

**心理测试概述**

心理测试包括两个部分：

1. **MBTI（迈尔斯-布里格斯性格类型指标）**
   - **测试链接**：[https://www.apesk.com/mbti](https://www.apesk.com/mbti)
   - **特点**：
     - 基于荣格的心理类型理论，评估个体在信息处理、决策方式、人际互动偏好等方面的性格特点。
     - 可能用于分析学生是否适合从事科研工作，但其效力尚不明确。

2. **MMPI（明尼苏达多项人格问卷）**
   - **测试链接**：[http://www.apesk.com/mmpi](http://www.apesk.com/mmpi)
   - **特点**：
     - 主要用于评估学生的心理健康状态，排查潜在精神异常问题。
     - 可能被用作衡量学生是否可能对研究机构构成安全隐患的辅助工具，其结果效力也尚不明确。

2. **综合面试**：用文献阅读时制作的PPT进行汇报，汇报时间每人15分钟，包含10分钟报告和5分钟提问。第1分钟需用英文汇报，后续可中英文混合。<br>

3. **个人展示**：提前制作个人展示PPT，汇报内容包括个人基本情况、主要课程成绩、排名、外语成绩、获奖荣誉、科研经历及其他相关信息。每人展示时间15分钟，报告10分钟，提问5分钟。前2分钟需进行英文自我介绍，提问环节中会有英文听说能力的考察。<br>
<span style="color: orange;">在综合面试阶段和个人展示阶段会穿插问你专业知识，千万要复习！我参加时候以为完全不会问就完全没准备，一问一个不吱声。个人展示阶段阶段会抓着你的科研经历问专业知识，当然综合面试阶段老师们问的闲聊问题的含量会很多。着重复习物理海洋学，有条件尽量学习气候动力学、大气辐射学、大气物理学。</span> <br>

4. **师生交流**：这个部分是互选的辅助阶段，在交流时注意导师态度，及时调整志愿顺序。<br>

  <h2>5. 优秀营员评选机制</h2>

  <p>优秀营员的评选是大气与海洋科学系夏令营的关键环节，其评选机制结合了弱委员会机制（weak committee-driven）和强委员会机制（strong committee-driven）的特性。评选过程由全系教师共同讨论，综合考虑学生在综合面试和个人展示阶段的表现，并依据学生填报的志愿顺序进行匹配。</p>

  <h3>评选流程</h3>
  <ol>
    <li>
      <strong>入选名单的确定</strong><br>
      根据综合面试和个人展示的评分结果，按照录取名额的120%确定优秀营员的入选名单。入选名单中的学生具备进一步参与双向选择的资格，最终排名靠前的100%学生将获得正式录取资格。
    </li>
    <li>
      <strong>双向选择机制</strong><br>
      在第一轮选择中，学生的排名与志愿顺序起决定性作用，排名靠前的学生通常优先被第一志愿导师录取。这一阶段主要体现弱委员会机制的特性，即导师在选择学生时拥有较强的自主性，更多依据个人意愿决定录取对象，而委员会的作用相对较弱。
    </li>
    <li>
      <strong>调剂程序</strong><br>
      若第一轮双向选择后仍有导师未完成招生目标，则进入调剂阶段。在这一阶段，导师的选择不仅受个人意愿驱动，还需要通过委员会的多轮讨论和协调以确定录取人选。此过程体现了强委员会机制的特点，即委员会在录取决策中的作用显著增强，通过集体协商确保招生工作的公平性与合理性。
    </li>
    <li>
      <strong>教务确认</strong><br>
      在调剂环节中，教务部门将在最终决策后及时联系相关学生（通常为下午电话通知），确认调剂结果与录取意向。
    </li>
  </ol>

  <h3>评选意义</h3>
  <p>
    这一评选机制平衡了导师自主选择权与委员会监督作用的重要性，同时强调学生的学术能力（硬实力）与前期沟通表现（软实力）的综合影响。学生在准备过程中，需注重提升自身学术能力、优化个人展示内容，同时积极与目标导师进行联系，确保对其研究方向有充分理解并展现出匹配性。这种综合评选模式既能保证招生工作的效率，也提升了录取结果的公平性和科学性。
  </p>
